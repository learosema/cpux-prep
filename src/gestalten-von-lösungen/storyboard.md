# Storyboard

> Eine Folge visueller Bildschirminhalte, die das Zusammenspiel zwischen einem Benutzer
> und einem interaktiven System veranschaulicht.

Ein Storyboard ist eine Comic-artige Darstellung eines Nutzungsszenarios.
Der Zweck eines Storyboards ähnelt dem Zweck eines Nutzungsszenarios.

Für ein Beispiel: siehe UXQB Curriculum, Seite 40.
